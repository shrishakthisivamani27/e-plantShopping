# e-plantShopping

## Paradise Nursery Shopping Application

Paradise Nursery is a React and Redux-based shopping application that allows users to browse and purchase a variety of houseplants online. The application provides an interactive shopping experience with dynamic cart management and real-time updates using Redux Toolkit.

## Features

- Browse plants grouped by categories
- View plant images, descriptions, and prices
- Add plants to the shopping cart (once added , changed to added to cart)
- Automatically update cart item count
- Increase or decrease item quantity
- Remove items from the cart
- Calculate total cost dynamically
- Continue shopping from the cart page
- Checkout button for future enhancements
- Responsive and user-friendly interface

## Technologies Used

- React
- Redux Toolkit
- React Redux
- JavaScript 
- CSS3
- Vite

## Project Structure

- `AboutUs.jsx` – Company information and introduction
- `ProductList.jsx` – Plant catalog and shopping functionality
- `CartItem.jsx` – Shopping cart management
- `CartSlice.jsx` – Redux state management for cart operations
- `store.js` – Global Redux store configuration
